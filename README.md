# — CRM / CLIENT MANAGEMENT DASHBOARD UI

Build a complete modern SaaS CRM dashboard UI for a fictional digital agency called “NovaFlow Agency”.

The CRM dashboard should feel like a real-world B2B SaaS product used daily by a small agency team managing:

* leads
* clients
* follow-ups
* projects
* internal tasks

This is NOT a generic dashboard concept.

The UI should feel realistic, scalable, and production-ready — something that could actually be used by a real agency.

The design must prioritize:

* workflow efficiency
* usability
* scalability
* clarity
* information hierarchy
* productivity

---

## BUSINESS SCENARIO

Design this CRM for:
“A 5-member digital marketing and web design agency managing 30+ active clients and incoming leads.”

Users of the dashboard include:

* agency owner
* sales manager
* account managers
* team members

The dashboard should support:

* lead tracking
* client management
* follow-up planning
* task organization
* engagement overview
